================================ IMP POINTS =================================
1.List
-- List is mutable
-- List is declared using []
-- Elements in list can be accessed through []
-- Contains different data types in single list
-- List has dynamic memory
-- Allow duplicate elements
-- Indexing is available

2. Tuple
-- Tuples are immutable
-- tuples are declared using ()
-- Elements in tuple are accessed using []
-- Tuple can have different data types in it
-- We cannot add or append elements in tuple as tuple is immutable
-- Tuple has static memory
-- Tuple allow duplicates
-- Indexing is available

3. set
-- Set is mutable
-- Set is declared using {}
-- Elements is set cannot be accessed as it is unordered collection
-- Set allow different data types in it
-- Set does not allow duplicate elements
-- indexing is not available


 
4. Range
-- Range is not mutable as it is a sequence of numbers
-- Range is declared using()
-- Used to declare range between two numbers
-- Range is not mutable
-- As its a sequence no duplication
-- Indexing is allowed

5. Dictionary
-- Dictionary is a key and value pair declaration
-- Dictionary is declared using{key,value}
-- Key is unique and value may be duplicate
-- We can change the key and values, its mutable
-- Elements are accessed by key
-- Dictionary used dynamic memory allocation.
